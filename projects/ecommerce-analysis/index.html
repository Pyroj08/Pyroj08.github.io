E-Commerce Data Analysis Project

An end-to-end case study using SQL, Python, and Power BI to transform raw e-commerce sales data into actionable insights and interactive dashboards.

Project Overview

This project demonstrates the complete workflow of e-commerce data analysis:

  • Imported raw CSV files into a MySQL database.
  • Cleaned and transformed data in Python (Pandas & SQLAlchemy).
  • Stored the refined dataset back into MySQL for structured access.
  • Built interactive dashboards in Power BI to highlight sales, customers, and product insights.

The dashboard provides a clear picture of customer behavior, sales trends, product performance, and fulfillment metrics.
