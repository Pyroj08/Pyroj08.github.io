E-Commerce Data Analysis

This project demonstrates a complete end-to-end **E-Commerce Data Analytics workflow**. Raw data was first imported into MySQL, cleaned using Python and SQLAlchemy, then re-stored in MySQL as a single consolidated dataset. Finally, the cleaned data was connected to **Power BI** to build interactive dashboards and extract meaningful insights on sales, customer behavior, and product performance.

Database Schema

The schema outlines the relationships among orders, customers, payments, and product data, providing a structured foundation for efficient queries and downstream analysis.
