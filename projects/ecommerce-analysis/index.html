Project Overview

This project analyzes raw e-commerce transaction data, transforming it into meaningful insights through a structured ETL process:

  • Raw data was first imported into a MySQL database.
  • Data cleaning and preprocessing were performed in Python (Pandas & SQLAlchemy).
  • Cleaned datasets were stored back in MySQL as structured tables.
  • Power BI was connected to the processed database for interactive dashboards and insights.

This pipeline ensures reliability, scalability, and clarity in extracting actionable business insights.

Database Schema

The schema diagram illustrates relationships between tables in the cleaned database.

Power BI Dashboard

The dashboard provides interactive visualizations of sales trends, customer behavior, and product performance.

Key Insights

  • Top-performing product categories and sellers identified.
  • Seasonal and regional purchase trends tracked.
  • Customer segmentation done using RFM analysis.
  • Delivery performance metrics highlighted for business optimization.
