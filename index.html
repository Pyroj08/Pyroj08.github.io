E-Commerce Data Analysis Project

An end-to-end data pipeline project covering SQL, Python, and Power BI to transform raw data into actionable insights.

Project Overview

This project demonstrates the complete workflow for analyzing e-commerce sales data:

  • Imported raw CSV files into MySQL database.
  • Cleaned & transformed data in Python (using Pandas & SQLAlchemy).
  • Stored cleaned dataset back into MySQL for structured access.
  • Built an interactive Power BI dashboard to generate insights.

Key Insights

  • Sales peak during holiday seasons with spikes in Q4.
  • Top 10 customers contribute significantly to revenue.
  • Credit card payments dominate overall transactions.
  • Delivery delays are mostly linked to specific regions.

Tech Stack

  • SQL (MySQL): Database design, storage & queries
  • Python: Data cleaning, transformation, and EDA
  • Pandas, SQLAlchemy: Data manipulation and integration
  • Power BI: Dashboard creation & visualization
